			      << DESCRIPTION >>
---------------------------------------------------------------------------
This system serves the purpose of generating possible timetables based on
user's candidate courses with descriptive constraints, such as what is the
earliest time of the first lecture/tutorial and day-off on a specific day.
With all the data and constraints, the system provides users a possible
timetable which meets their requirements with a list of CRNs required for
course registration.
